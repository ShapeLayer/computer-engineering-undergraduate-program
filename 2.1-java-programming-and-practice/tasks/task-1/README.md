## 3 [교재 110p 1번문제] 
Scanner 클래스를 이용하여 원화를 입력받아 달러로 바꾸어 다음 예시와 같이 출력하는 프로그램을 작성하라. $1 = 1100.00 로 가정하고 계산하라. 

실행예
```
원화를 입력하세요 (단위 원) >> 3300 
3300원은 $3.0 입니다. 
```

## 4.  
Scanner 클래스를 이용하여 제곱미터 ($m^2$) 을 입력받아 몇 평인지 출력하는 프로그램을 작성하라. 
(단 1평은 3.305785 $m^2$ 로 계산한다) 

실행예
```
제곱미터를 입력하세요 >> 128 
128m^2은 38.72평 입니다. 
```

## 5. [교재 110p 2번문제]

실행예
```
2자리수 정수 입력 (10~99) >> 77 
Yes! 10의자리와 1의자리가 같습니다. 
```
