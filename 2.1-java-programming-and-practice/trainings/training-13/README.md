## 실습과제 1 [교재 384].

중심을 나태내는 정수 x,y와 반지름 radius 필드를 가지는 Circle 클래스를 작성하고자 한다. 생성자는 3개의 인자 (x,y,radius)를 받아 해당 필드를 초기화 하고, equals() 메소드는 두개의 Circle 객체의 중심이 같으면 같은 것으로 판별하도록 한다. 

실행 예
```
원 a : Circle(2,3)반지름5 
원 b : Circle(2,3)반지름30 
같은 원 
```
