# 문제 2: 트리에서 소포 배달

깊이 4(루트 깊이 0, 최대 깊이 3)인 완전 이진트리를 생각한다.
노드는 총 15개이며, 각 노드는 `hasPackage`(불리언)를 가진다.
초기에는 모든 노드가 `hasPackage = false`이다.

입력으로 주어지는 `K`개의 노드에 소포를 배치(`hasPackage = true`)한 뒤,
출발 노드 `a`에서 도착 노드 `b`까지 이동한다.

단, 이동 경로에서 소포가 있는 모든 노드를 최소 1회 이상 방문해야 한다.
방문이란 해당 노드에 도달하는 것을 의미하며, 경로 중 통과도 방문으로 인정한다.

목표는 다음을 구하는 것이다.
- 소포 수 `k` (= `K`)
- 소포 노드를 모두 방문하면서 `a -> b`로 가는 최소 이동 횟수

이동 횟수는 부모-자식 간선 하나를 지날 때마다 1 증가한다.

## 노드 표기

노드는 정수 인덱스가 아니라 루트 기준 경로 문자열로 지정한다.

- `ROOT`: 루트
- `L`: 왼쪽 자식으로 이동
- `R`: 오른쪽 자식으로 이동
- 길이 1~3의 `L/R` 문자열 사용 가능 (예: `L`, `RR`, `LLR`, `RRL`)

## 입력

```text
K
path_1
path_2
...
path_K
a b
```

제약:
- `0 <= K <= 3`
- `path_i`, `a`, `b`는 `ROOT` 또는 길이 1~3의 `L/R` 문자열
- `path_i`는 서로 다름

## 출력

- `k = 0`이면 `0`만 출력
- `k > 0`이면 아래 2줄 출력

```text
k
minMoves
```

## 입출력 예 1

입력:

```text
3
ROOT
LR
RRL
LLL LLL
```

출력:

```text
3
14
```

## 입출력 예 2

입력:

```text
3
LLL
RRR
LR
LLR RLL
```

출력:

```text
3
14
```

## 입출력 예 3

입력:

```text
3
LRL
RRL
LL
LRL RRL
```

출력:

```text
3
8
```

## 입출력 예 4

입력:

```text
1
ROOT
LLL RRR
```

출력:

```text
1
6
```
