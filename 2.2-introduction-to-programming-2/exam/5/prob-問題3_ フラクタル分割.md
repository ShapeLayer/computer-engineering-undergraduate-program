# 문제 3: 프랙탈 분할

한 변 길이가 `W`인 정사각형 캔버스를 생각한다.
좌표계는 다음과 같다.

- 좌상단: `(0, 0)`
- 오른쪽으로 갈수록 `x` 증가
- 아래로 갈수록 `y` 증가
- 좌표 범위: `0 <= x, y < W`

`W = 3^N (N >= 1)`를 만족한다고 가정한다.
초기 상태에서 모든 픽셀은 검은색이다.

다음 분할 연산을 총 `N`회 수행한다.

## 분할 연산 (1회)

분할 시작 시점에 남아 있는 모든 정사각형 블록에 대해 동시에:

1. 각 블록을 `3 x 3`으로 등분
2. 가운데 블록(2행 2열)에 해당하는 픽셀을 제거
3. 한 번 제거된 픽셀은 이후 복구되지 않음

## 출력

각 단계 `i (1 <= i <= N)`에 대해,
"그 단계에서 새로 제거된 픽셀"의 좌표만 다음 형식으로 한 줄 출력한다.

```text
i=i:(x1,y1),(x2,y2),...,(xK,yK)
```

정렬 규칙:
- `y` 오름차순
- `y`가 같으면 `x` 오름차순

좌표 사이 구분자는 쉼표만 사용하고 공백은 넣지 않는다.

## 입력

```text
W
```

- `W = 3^N`, `N >= 1`

## 입출력 예 1

입력:

```text
3
```

출력:

```text
i=1:(1,1)
```

## 입출력 예 2

입력:

```text
9
```

출력:

```text
i=1:(3,3),(4,3),(5,3),(3,4),(4,4),(5,4),(3,5),(4,5),(5,5)
i=2:(1,1),(4,1),(7,1),(1,4),(7,4),(1,7),(4,7),(7,7)
```

## 입출력 예 3

입력:

```text
27
```

출력:

```text
(길어서 생략)
```

## 입출력 예 4

입력:

```text
81
```

출력:

```text
(매우 길어서 생략)
```
