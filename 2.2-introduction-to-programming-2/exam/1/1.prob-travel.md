# 문제 1: 전국 여행 지원

일본의 관광 수요를 촉진하기 위해 "전국 여행 지원" 캠페인이 진행되고 있다.
아래 규칙에 따라 여행 대금을 할인하고, 지역 쿠폰을 지급한다.

숙박일수, 여행 대금, 교통 포함 여부, 요일 정보를 입력받아
할인 후 여행 대금과 지급되는 지역 쿠폰 금액을 출력하는 프로그램을 작성하라.

## 입력

다음 값을 순서대로 입력한다.

```text
숙박일수
여행 대금
교통 포함 여부 (1: 포함, 0: 미포함)
요일 정보
```

## 출력

다음 형식으로 출력한다.

```text
숙박일수: 여행대금: 교통(1:유, 0:무): 요일
금액: <할인 후 여행 대금> 엔
지역 쿠폰 지급액: <쿠폰 금액> 엔
```

## 규칙

1. 할인액은 원래 여행 대금의 20%이다.
2. 할인액의 1엔 미만은 버린다(절사).
3. 1박당 할인 상한:
	- 교통 포함 여행: 1박당 최대 5000엔
	- 교통 미포함 여행: 1박당 최대 3000엔
4. 지역 쿠폰 지급액:
	- 평일 숙박: 1박당 2000엔
	- 휴일(일요일) 숙박: 1박당 1000엔
5. 단, 할인 후 여행 대금이 아래 기준보다 작으면 캠페인 대상 외로 간주한다.
	- 기준값 = (평일 숙박 수 x 3000엔) + (휴일 숙박 수 x 2000엔)
	- 이 경우 할인액은 0엔, 지역 쿠폰 지급액도 0엔으로 한다.

## 입출력 예 1

입력:

```text
2
55000
0
2
```

출력:

```text
숙박일수: 여행대금: 교통(1:유, 0:무): 요일
금액: 49000 엔
지역 쿠폰 지급액: 4000 엔
```

## 입출력 예 2

입력:

```text
2

0
2
```

출력:

금액: 4500 엔
지역 쿠폰 지급액: 0 엔
