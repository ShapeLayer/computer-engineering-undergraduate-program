# 문제 7: 문자열 사전 버전 관리

파일 이름과 버전 번호를 관리하는 시스템을 구현하라.
아래 명령을 순서대로 처리한다.

- `s`: 등록(set)
- `r`: 삭제(remove)
- `q`: 접두사 검색(query)
- `e`: 종료(exit)

내부 저장소는 `map<string, int>`를 사용한다고 가정한다.

## 명령 사양

### 1) 등록 `s name version`

- `name`은 영문자(`A-Z`, `a-z`)만 허용
- 영문자 외 문자가 하나라도 있으면:

```text
오류: 파일 이름은 영문자만 사용 가능
```

- 정상 입력이면 `name -> version` 저장(기존 이름이 있으면 덮어씀)

### 2) 삭제 `r name`

- 해당 이름이 없으면:

```text
오류: 이 파일이 없습니다.
```

- 있으면 삭제

### 3) 검색 `q prefix`

- 현재 등록된 파일 중 `prefix`로 시작하는 파일들을 찾는다.
- 그중 최대 버전을 출력한다.
- 해당 파일이 없으면:

```text
오류: 이 파일이 없습니다.
```

- 있으면:

```text
최대 버전: X
```

### 4) 종료 `e`

- 다음 메시지를 출력하고 명령 처리를 끝낸다.

```text
조작 종료
```

- 이어서 아래를 출력한다.

```text
등록된 파일 목록:
```

- 현재 등록 파일명을 사전순 오름차순으로 한 줄씩 출력

## 입력

여러 줄에 걸쳐 명령이 주어지며 `e`가 나올 때 종료한다.

```text
s name version
r name
q prefix
e
```

## 출력

- 매 명령 처리 전, 아래 프롬프트를 출력:

```text
명령 입력(s: 등록 / r: 삭제 / q: 검색 / e: 종료)
```

- 명령 처리 결과(오류/검색 결과/종료 후 목록)를 사양대로 출력

## 입출력 예 1

입력:

```text
s apple 3
s banana 5
q a
e
```

핵심 출력:

```text
최대 버전: 3
조작 종료
등록된 파일 목록:
apple
banana
```

## 입출력 예 2

입력:

```text
s app1e 3
s cat 4
r dog
q z
e
```

핵심 출력:

```text
오류: 파일 이름은 영문자만 사용 가능
오류: 이 파일이 없습니다.
오류: 이 파일이 없습니다.
조작 종료
등록된 파일 목록:
cat
```

## 입출력 예 3

입력:

```text
s alpha 1
s alpine 7
s alps 4
r alpha
q alp
e
```

핵심 출력:

```text
최대 버전: 7
조작 종료
등록된 파일 목록:
alpine
alps
```
