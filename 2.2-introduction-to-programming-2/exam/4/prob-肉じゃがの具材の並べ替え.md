# 문제 2: 니쿠자가 재료 순서 바꾸기

니쿠자가는 재료를 넣는 순서에 따라 맛이 달라진다.
초기 재료 순서는 다음과 같다.

1. 양파
2. 고기
3. 감자
4. 당근
5. 설탕
6. 간장
7. 미림

입력으로 재료 이름 두 개 `s1`, `s2`가 주어진다.
초기 순서에서 `s1`과 `s2`의 위치를 서로 바꾼 뒤, 최종 순서를 위에서부터 한 줄에 하나씩 출력하라.

단, `s1 == s2`이면 순서는 바뀌지 않는다.

## 입력

```text
s1 s2
```

`s1`, `s2`는 다음 중 하나:

- 양파
- 고기
- 감자
- 당근
- 설탕
- 간장
- 미림

## 출력

- 최종 재료 순서를 7줄로 출력
- 각 줄에는 재료명만 출력

## 입출력 예 1

입력:

```text
양파 감자
```

출력:

```text
감자
고기
양파
당근
설탕
간장
미림
```

## 입출력 예 2

입력:

```text
고기 간장
```

출력:

```text
양파
간장
감자
당근
설탕
고기
미림
```

## 입출력 예 3

입력:

```text
당근 당근
```

출력:

```text
양파
고기
감자
당근
설탕
간장
미림
```
