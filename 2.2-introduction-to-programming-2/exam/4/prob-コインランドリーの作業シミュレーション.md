# 문제 6: 코인세탁소 작업 시뮬레이션

코인세탁소에서 고객은 다음 3단계를 거친다.

1. 세탁
2. 건조
3. 접기

세탁기는 `W`대, 건조기는 `D`대, 접기 작업대는 1대다.

각 고객에 대해 `접기 완료 시간 - 세탁 시작 시간`을 계산하고,
모든 고객의 값을 합산해 출력하라.

## 처리 규칙

### 세탁 단계

- 고객은 도착 시간에 세탁 대기열에 들어간다.
- 세탁기가 비고 대기열에 사람이 있으면, 대기열 앞에서부터 배정한다.
- 세탁이 끝나면 건조 대기열로 이동한다.

### 건조 단계

- 건조기도 동일하게 대기열 앞에서부터 배정한다.
- 건조가 끝나면 접기 대기 스택에 쌓는다.

### 접기 단계

- 접기 작업대가 비어 있고 스택이 비어 있지 않으면,
  스택의 맨 위 고객(가장 최근 건조 완료 고객)부터 처리한다. (LIFO)

## 입력

```text
N W D
arrival_0 wash_0 dry_0 fold_0
arrival_1 wash_1 dry_1 fold_1
...
arrival_{N-1} wash_{N-1} dry_{N-1} fold_{N-1}
```

- `1 <= N <= 100`
- `1 <= W, D <= 100`
- `arrival_i >= 0`
- `wash_i, dry_i, fold_i >= 1`

## 출력

- 아래 값을 정수로 출력

```text
sum(접기 완료 시간 - 세탁 시작 시간)
```

## 입출력 예 1

입력:

```text
3 2 1
0 3 4 2
0 3 4 2
1 3 4 2
```

출력:

```text
36
```

## 입출력 예 2

입력:

```text
3 1 2
0 2 1 6
1 2 2 12
1 1 1 4
```

출력:

```text
41
```

## 입출력 예 3

입력:

```text
3 3 2
0 3 1 2
2 2 1 3
3 1 1 4
```

출력:

```text
23
```
