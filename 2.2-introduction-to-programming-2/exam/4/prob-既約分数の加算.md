# 문제 3: 기약분수의 합

두 분수 `(a/b)`, `(c/d)`가 주어진다.
이 둘을 더한 결과를 기약분수로 출력하라.

이 문제에서는 함수 `gcd`와 `calc`를 구현하는 것이 목표이다.

- `gcd(x, y)`: 두 정수의 최대공약수 반환
- `calc(a, b, c, d)`: `(a/b) + (c/d)`를 계산해 기약분수 `(p/q)` 반환

추가 제약:

- `gcd`와 `calc` 구현에서 `for`, `while` 사용 금지
- 재귀 호출 사용 가능

## 입력

```text
a b c d
```

- `1 <= a, b, c, d <= 10^9`
- 중간 계산 및 최종 결과는 `long long` 범위 안에 있다고 가정

## 출력

- 결과 분모 `q != 1`이면 `p/q` 형식으로 출력
- 결과 분모 `q == 1`이면 정수 `p`만 출력

## 입출력 예 1

입력:

```text
50 100 75 100
```

출력:

```text
5/4
```

## 입출력 예 2

입력:

```text
5 6 7 6
```

출력:

```text
2
```

## 입출력 예 3

입력:

```text
8 9 10 27
```

출력:

```text
34/27
```
