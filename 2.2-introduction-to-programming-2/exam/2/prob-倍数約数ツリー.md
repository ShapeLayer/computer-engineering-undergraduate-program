# 문제 3 (배수/약수 트리)

정수 2개 A, B(1~10000)를 입력받아,
주어진 규칙으로 이진 트리를 구성해 출력하는 프로그램을 작성하라.

트리 구성 규칙은 다음과 같다.

1. 깊이 1의 루트 노드 값은 0이다.
2. 깊이 2의 왼쪽 노드에 A, 오른쪽 노드에 B를 배치한다.
3. 이후 각 깊이에서, 같은 깊이에 존재하는 노드들을 왼쪽부터 순서대로 쌍으로 묶어 아래 처리를 수행한다.
	1. 해당 쌍의 왼쪽 노드 값을 a, 오른쪽 노드 값을 b라 한다.
	2. a와 b의 최소공배수 LCM(a,b)를 구해,
		- 왼쪽 노드의 왼쪽 자식에 LCM(a,b)
		- 왼쪽 노드의 오른쪽 자식에 2*LCM(a,b)
		를 설정한다.
	3. a와 b의 최대공약수 GCD(a,b)를 구하고, 그 약수들을 큰 순서로 정렬한 배열을 만든다.
	4. 가장 큰 약수를 오른쪽 노드의 왼쪽 자식에,
		두 번째로 큰 약수를 오른쪽 노드의 오른쪽 자식에 설정한다.
		단, 약수가 부족하면 0으로 채운다.
4. 위 과정을 깊이 5까지 반복한다.

최종적으로 트리의 각 노드 값을 레벨 순서(깊이 순서)로 출력한다.
각 레벨은 1줄씩 출력하고, 노드 값은 공백으로 구분한다.

단, 정답 작성 구역 이외의 코드는 일절 수정하지 말 것.

## 입력

한 줄에 정수 A, B가 주어진다.

```text
A B
```

## 기대 출력

먼저 다음 문구를 출력한다.

```text
2 つの整数を入力してください (1 ~ 10000):
```

이어서 트리의 레벨 순 노드 값을 출력한다.
레벨마다 개행(endl)을 넣고, 출력 끝에 불필요한 공백/개행을 넣지 않는다.

## 입출력 예

입력:

```text
24 18
```

```text

0

72 144 6 3
144 288 72 36 6 12 3 1
288 576 144 72 72 144 36 18 12 24 6 3 3 6 1 0
```

입력:

```text
8 12
```

출력:

```text

0
```
