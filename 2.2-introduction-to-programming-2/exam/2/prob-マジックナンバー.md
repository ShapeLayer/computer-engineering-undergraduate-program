# 문제 5 (매직 넘버)

N개 팀이 참가하는 풀리그(총당리그)가 진행 중이다.
각 팀은 다른 팀과 2경기씩 대전하며, 결과는 승/패/무 중 하나이다.

현재 리그 중간 결과로 각 팀의
"승수(W), 패수(L), 무승부 수(T)"가 주어진다.
이를 바탕으로 선두 팀(승수가 가장 많은 팀)과
그 팀의 매직 넘버를 구하는 프로그램을 작성하라.

단, 정답 작성 구역 이외의 코드는 일절 수정하지 말 것.

매직 넘버는 선두 팀이 우승을 확정하기 위해 필요한 승수이다.
다른 팀이 남은 경기를 모두 이긴다고 가정해도,
선두 팀이 해당 수만큼 이기면 다른 팀보다 앞서는 상태가 되는 값이다.

승수가 같은 팀이 여러 개라면,
인덱스가 더 작은 팀을 선두 팀으로 한다.

또한 입력값에 모순이 있는 경우(총 경기 수, 승패 정합성 불일치 등)
표준 에러 출력으로 "입력 오류"를 출력하고 종료한다.

입력은 다음 조건을 모두 만족해야 한다.

- 각 팀의 총 경기 수 (Wi + Li + Ti + Ri)는 G = 2*(N-1)을 초과할 수 없다.
- 리그 전체 총 승수 ΣW 와 총 패수 ΣL 은 일치해야 한다.
- 남은 경기 합 ΣR 은 짝수여야 한다.
- 가장 많이 남은 팀의 경기 수 maxR 은 전체 남은 경기의 절반(ΣR/2) 이하여야 한다.

## 입력

```text
N
W_1 L_1 T_1
W_2 L_2 T_2
...
W_N L_N T_N
```

- N: 팀 수 (2 <= N <= 10)
- W_i: 팀 i의 현재 승수
- L_i: 팀 i의 현재 패수
- T_i: 팀 i의 현재 무승부 수

## 기대 출력

다음 3가지 중 하나의 형식으로 출력하고,
마지막에 구분선(----------------------------------------)을 출력한다.

1) 매직 넘버가 점등된 경우

```text
----------------------------------------
선두 팀 : X ( 승리수 : W_top, 남은 경기 : R_top)
매직 넘버 : M
----------------------------------------
```

2) 매직 넘버가 미점등인 경우

```text
----------------------------------------
선두 팀 : X ( 승리수 : W_top, 남은 경기 : R_top)
매직 넘버는 점등하지 않았습니다 (필요 승수 M이 남은 경기 수 R_top을 초과)
----------------------------------------
```

3) 선두 팀 우승이 확정되어 타 팀의 자력 우승이 불가능한 경우

```text
----------------------------------------
선두 팀 : X ( 승리수 : W_top, 남은 경기 : R_top)
매직 넘버는 점등하지 않았습니다 (타 팀 자력 우승 소멸)
----------------------------------------
```

## 입출력 예

입력:

```text
3
2 0 0
1 1 0
0 2 0
```

출력:

```text
----------------------------------------
선두 팀 : 1 ( 승리수 : 2, 남은 경기 : 2)
매직 넘버 : 2
----------------------------------------
```

입력:

```text
3
1 0 0
2 0 0
0 3 0
```

출력:

```text
----------------------------------------
선두 팀 : 2 ( 승리수 : 2, 남은 경기 : 2)
매직 넘버는 점등하지 않았습니다 (필요 승수 3이 남은 경기 수 2를 초과)
----------------------------------------
```
