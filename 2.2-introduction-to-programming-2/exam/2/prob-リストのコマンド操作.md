# 문제 4 (리스트의 명령 조작)

영숫자로 구성된 특수 명령 문자열이 주어진다.
이 문자열을 바탕으로 정수 값을 노드로 갖는 단방향 리스트를 구성/조작하고,
최종 리스트 내용을 `->` 구분자로 출력하라.
출력의 맨 끝에는 `->`를 붙이지 않는다.

단, 정답 작성 구역 이외의 코드는 일절 수정하지 말 것.

## 명령 문자열 규칙

- 숫자(`0`~`9`): 해당 값을 리스트에 추가
- `a`: 이후 숫자들을 오름차순 정렬 대상 구간으로 처리
- `d`: 이후 숫자들을 내림차순 정렬 대상 구간으로 처리
- `f`: 직후에 이어지는 숫자들을 소수 삭제 대상 구간으로 처리

## 동작 규칙

- 명령이 없는 숫자는 그대로 리스트의 끝에 추가한다.
- `a` 모드에서는 입력 종료 또는 다음 `a`/`d`가 나올 때까지를 하나의 구간으로 본다.
- 이 구간 안에 `f`가 있으면, `f` 이전(비 `f` 구간)과 `f` 이후(`f` 구간)을 구분해 처리한다.
- `f` 구간에 포함된 한 자리 소수(`2, 3, 5, 7`)를 삭제한 뒤, 비 `f` 구간과 연결하고, 구간 전체를 오름차순 정렬해 출력 리스트에 연결한다.
- `d` 모드도 동일하게 하나의 구간을 취한다.
- `f`가 없으면 구간 전체를 내림차순 정렬한다.
- `f`가 있으면 비 `f` 부분과 `f` 부분을 각각 독립적으로 내림차순 정렬한 뒤, `비 f -> f` 순서로 연결하여 출력 리스트에 붙인다.
- 입력 끝에서 모드 구간이 남아 있어도 위 규칙으로 처리하여 출력 리스트에 추가한다.

## 입력

- 한 줄의 명령 문자열 (예: `12a34f56d789`)

## 기대 출력

- `->`로 연결된 리스트 (예: `1->2->3->4->5`)

## 입출력 예

입력 | 출력
---|---
`5a1f43d17f12` | `5->1->4->7->1->2->1`
`2a5314` | `2->1->3->4->5`

## 처리 흐름 예 (입력: `5a1f43d17f12`)

입력 구간 | 처리 내용
---|---
`5` | 모드 없음 -> 그대로 출력 대상에 추가
`a1f43` | 오름차순 모드 시작, `f` 구간의 `4,3`에서 소수 `3` 삭제 -> `[1,4]` 오름차순
`d17f12` | 내림차순 모드 시작, 구간 `[1,7,f,1,2]`, `f` 밖 `[1,7] -> [7,1]`, `f` 안 `[1,2] -> [2,1]`, 결합 `[7,1,2,1]`
