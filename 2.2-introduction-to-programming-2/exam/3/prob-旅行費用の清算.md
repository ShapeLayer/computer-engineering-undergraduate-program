# 문제 2: 여행 비용 정산

친구들과 여러 번의 여행을 갔고, 매 여행마다 한 사람이 대표로 비용을 결제했다.
여행이 끝난 뒤 모두가 공평하게 부담하도록, 누가 누구에게 얼마를 보내야 하는지 계산하는 프로그램을 작성하라.

## 입력

- 첫 줄: 여행 횟수 `days`
- 둘째 줄: 참가자 수 `people`
- 이후 `days`줄: 각 여행의 결제 정보
	- `cost payer`
	- `cost`: 해당 여행의 총 결제 금액
	- `payer`: 결제한 사람 번호(1-based)

## 출력

아래 형식으로 출력한다.

```text
총비용: <값> 엔
1인당 부담액: <값> 엔
각 사람의 정산 차액:
1번: +<값> 엔 / -<값> 엔 / 0 엔
2번: ...
...
지불 내역:
A번 -> B번: <값> 엔
```

- `지불 내역`은 부족한 사람이 초과 지불한 사람에게 보내는 방식으로 구성한다.
- 모두 차액이 0이면 `지불 내역` 아래에는 아무 줄도 출력하지 않는다.

## 입출력 예 1

입력:

```text
3
3
12000 1
18000 2
9000 3
```

출력:

```text
총비용: 39000 엔
1인당 부담액: 13000 엔
각 사람의 정산 차액:
1번: -1000 엔
2번: +5000 엔
3번: -4000 엔
지불 내역:
1번 -> 2번: 1000 엔
3번 -> 2번: 4000 엔
```
