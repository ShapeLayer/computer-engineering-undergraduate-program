# 문제 3: 도트 디자인 최대 영역 탐색

높이 `H`, 너비 `W`의 캔버스가 주어진다.
캔버스의 각 칸에는 `0~9`의 정수가 들어 있다.

같은 정수값이 상하좌우로 연결된 칸들의 집합을 하나의 영역으로 본다.
이때 값이 `0`인 칸은 배경으로 간주하며 영역 크기 계산에서 제외한다.

0이 아닌 값들로 이루어진 연결 영역들 중 가장 큰 영역의 칸 수를 구하라.

## 입력

- 첫 줄: `H W` (`1 <= H, W <= 20`)
- 다음 `H`줄: 각 줄에 `W`개의 정수(`0~9`)

## 출력

- 가장 큰 영역의 칸 수(정수)

## 입출력 예 1

입력:

```text
6 8
0 0 2 2 0 3 3 3
1 1 2 0 0 3 0 0
1 1 2 4 4 4 4 0
0 0 0 4 0 0 4 0
5 5 5 4 4 4 4 4
5 0 5 5 0 0 0 0
```

출력:

```text
11
```

## 입출력 예 2

입력:

```text
5 7
1 0 1 0 1 0 1
1 1 0 0 0 1 1
0 1 0 2 2 2 0
3 3 0 2 0 2 0
3 0 0 2 2 2 0
```

출력:

```text
8
```

## 입출력 예 3

입력:

```text
6 8
0 0 4 4 4 0 0 0
0 4 4 0 1 4 4 0
4 4 0 0 0 0 4 0
0 4 4 4 0 4 4 0
0 0 0 0 0 0 0 0
5 5 5 5 5 5 5 5
```

출력:

```text
10
```
