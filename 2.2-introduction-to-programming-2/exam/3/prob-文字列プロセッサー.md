# 문제 4: 문자열 프로세서

한 줄 문자열 `s`가 주어진다.
다음 규칙으로 문자열을 변환하고, 포함된 숫자들의 합을 16진수로 출력하라.

## 처리 규칙

- 소문자 알파벳은 대문자로 변환한다.
- 대문자 알파벳은 소문자로 변환한다.
- 숫자 문자는 값으로 합산한 뒤, 최종 문자열에서는 제거한다.
- 알파벳/숫자 이외 문자는 그대로 유지한다.
- 마지막 줄에 합산한 숫자의 합을 16진수(대문자)로 출력한다.

## 입력

- 한 줄 문자열

## 출력

1. 변환된 문자열(숫자 제거 후)
2. 숫자 합의 16진수 표현

## 입출력 예

입력:

```text
hello34WOr5ld
```

출력:

```text
HELLOwoRlD
C
```

입력:

```text
aaa111bbb222ccc333ddd444eee555
```

출력:

```text
AAABBBCCCDDDEEE
2D
```

입력:

```text
Ab9_cd88Ef!
```

출력:

```text
aB_CDeF!
19
```
