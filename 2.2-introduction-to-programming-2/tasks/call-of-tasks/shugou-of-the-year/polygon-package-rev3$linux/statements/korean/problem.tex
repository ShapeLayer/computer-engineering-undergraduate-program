\begin{problem}{Zaru of the Year}{standard input}{standard output}{10 seconds}{256 megabytes}

사가대학은 올해의 대학제 기간에, 학생들이 가져와 마시는 술과 그 학생들의 소속 학부 정보를 획득하였다. 이 정보를 적절히 정리하여, 섭취한 알코올의 총량이 가장 높은 학부의 이름을 알아내어라. 주어진 클래스에 선언된 메서드들을 적절히 구현하고, STL을 활용하여야 한다.

\InputFile
첫째 줄에 술을 가져와 마신 학생의 수 $N$이 주어진다. $(1 \le N \le 10^7)$
이어지는 $N$개 줄에 걸쳐, 각 학생의 소속 학부 $S$, 술의 용량(L) $V$, 알코올 도수 $P$가 공백으로 구분되어 주어진다. $(1 \le |S| \le 25, 1 \le V \le 1000, 0 \le P \le 100)$

$S$는 알파벳 소문자와 \tt{_}(언더바)로만 이루어진 문자열로, 항상 다음 중 한개임이 보장된다.

- \tt{education}

- \tt{science_and_engineering}

- \tt{economics}

- \tt{agriculture}

- \tt{art_and_regional_design}

$V$와 $P$는 정수이다.

\OutputFile
섭취한 알코올의 총량이 가장 높은 학부의 이름을 출력한다. 만약, 가장 높은 학부가 여러개라면, 사전순으로 가장 앞서는 학부의 이름을 출력한다.


\end{problem}
