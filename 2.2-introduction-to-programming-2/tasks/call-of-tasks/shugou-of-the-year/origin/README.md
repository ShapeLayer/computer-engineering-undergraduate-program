# Sulgorae of the Year
# Shugou of the Year

사가대학은 올해의 대학제 기간에, 학생들이 가져와 마시는 술과 그 학생들의 소속 학부 정보를 획득하였다. 이 정보를 적절히 정리하여, 섭취한 알코올의 총량이 가장 높은 학부의 이름을 알아내어라. 주어진 클래스에 선언된 메서드들을 적절히 구현하고, STL을 활용하여야 한다.

佐賀大学は、今年の大学祭の期間に、学生たちが持ち寄って飲んだお酒と、その学生たちの所属学部情報を入手しました。この情報を適切に整理し、摂取したアルコールの総量が最も多い学部の名前を特定してください。与えられたクラスに宣言されたメソッドを適切に実装し、STLを活用する必要があります。

## 입력

첫째 줄에 술을 가져와 마신 학생의 수 N이 주어진다. (1 <= N <= 1e6)
이어지는 N개 줄에 걸쳐, 각 학생의 소속 학부 S, 술의 용량(L) V, 알코올 도수 P가 공백으로 구분되어 주어진다. (1 <= |S| <= 25, 1 <= V <= 1000, 0 <= P <= 100)

S는 알파벳 소문자와 _(언더바)로만 이루어진 문자열로, 항상 다음 중 한개임이 보장된다.
- `education`
- `science_and_engineering`
- `economics`
- `agriculture`
- `art_and_regional_design`

V와 P는 정수이다.

1行目に、お酒を持ち寄って飲んだ学生の数 N が与えられます。(1 <= N <= 1e6)
続くN行にわたり、各学生の所属学部 S、お酒の容量 V、アルコール度数 P がスペース区切りで与えられます。(1 <= |S| <= 25, 1 <= V <= 1000, 0 <= P <= 100)

Sはアルファベットの小文字と _ (アンダースコア)のみで構成される文字列で、常に以下のいずれかであることが保証されます。

- `education`
- `science_and_engineering`
- `economics`
- `agriculture`
- `art_and_regional_design`

VとPは整数です。

## 출력

섭취한 알코올의 총량이 가장 높은 학부의 이름을 출력한다. 만약, 가장 높은 학부가 여러개라면, 사전순으로 가장 앞서는 학부의 이름을 출력한다.

摂取したアルコール総量が最も多い学部の名前を出力してください。総量が最多の学部が複数ある場合は、辞書順で最も先に来る学部の名前を出力してください。

## 예제 입력 1

```
5
science_and_engineering 500 10
education 300 20
economics 400 8
art_and_regional_design 200 10
science_and_engineering 600 10
```

## 예제 출력 1

```
science_and_engineering
```
